Operadores de comparacion
Ejemplos que dan true (verdadero):
console.log(5 == '5');     // true → compara solo valor
console.log(5 === 5);      // true → valor y tipo
console.log(3 < 10);       // true
console.log(8 > 1);        // true
console.log(4 <= 4);       // true

Ejemplos que dan false (falso):
console.log(5 === '5');    // false → tipos diferentes
console.log(10 != 10);     // false
console.log(7 < 3);        // false
console.log(4 > 9);        // false
console.log(2 >= 5);       // false
// ! esto es para negar el valor, o invertirlo.
